Simple Interest Calculator
This project is a simple web-based calculator that computes Simple Interest based on the principal amount, interest rate, and time.

Formula
SI = (P × R × T) / 100

Where:

P = Principal Amount
R = Rate of Interest
T = Time (years)
Technologies Used
HTML
JavaScript
How to Use
Enter the principal amount
Enter the interest rate
Enter the time in years
Click the Calculate button
The simple interest will be displayed on the screen.
