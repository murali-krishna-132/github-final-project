# Simple Interest Calculator

This project is a simple web-based calculator that computes **Simple Interest** based on the principal amount, interest rate, and time.

## Formula

SI = (P × R × T) / 100

Where:
- P = Principal Amount
- R = Rate of Interest
- T = Time (years)

## Technologies Used
- HTML
- JavaScript

## How to Use

1. Enter the principal amount
2. Enter the interest rate
3. Enter the time in years
4. Click the **Calculate** button
5. The simple interest will be displayed on the screen.
